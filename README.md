# Praktik Git Bash

Repository ini dibuat untuk memenuhi Praktik Git dan GitHub.

## Praktik 1

- Membuat repository Git
- Inisialisasi repository (`git init`)
- Membuat file README.md
- Melakukan commit pertama

## Praktik 2

- Mengubah isi README.md
- Melakukan `git add`
- Melakukan `git commit`
- Melihat riwayat commit

## Perintah yang digunakan

```bash
git init
git add .
git commit -m "Commit pertama"
git status
git log --oneline
```
